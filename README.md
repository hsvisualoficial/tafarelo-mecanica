# Tafarelo Mecânica — Landing Page

Site institucional da **Tafarelo Mecânica** (Jundiaí - SP) — oficina automotiva com 48 anos de tradição.

Página única, estática e responsiva (HTML/CSS/JS puro, sem frameworks).

## Estrutura
- `index.html` — página completa (todas as seções, CSS e JS inline)
- `assets/` — logos, imagens e ícones otimizados
- `robots.txt` / `sitemap.xml` — SEO

## Recursos
- Responsivo (mobile / tablet / desktop)
- SEO completo: meta tags, Open Graph, Twitter Card e Schema.org (LocalBusiness)
- Google Tag Manager (GTM-58PQNZTK)
- WhatsApp flutuante, formulário com envio para WhatsApp, carrosséis e animações
- Avaliações Google reais (4.7 ⭐ · 126 avaliações)

## Rodar localmente
```bash
npx serve .
```
